## Build a widget release
Add local changes with a commit message / new branch or main
Tag the vite project (patch/minor/major)
Create a live release of the tag using the command:
fromt he widget root folder: 
`node ../release/scripts/release.js <widgetname>`

```bash
npm create vite@latest vite_project (React/Typescript)
cd vite_project
npm install
```
